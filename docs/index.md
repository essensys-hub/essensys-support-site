# Documentation Essensys Support Site

Bienvenue sur la documentation technique du site de support Essensys.

Ce projet se compose de :
- Un **Backend** en Go (Golang) utilisant Chi Router.
- Un **Frontend** en React (Vite).
- Une infrastructure hébergée sur VPS (Ubuntu) avec Nginx et PostgreSQL.

## Navigation

- [Configuration Système](configuration.md) : Base de données, Backups, Variables d'environnement.
- [Authentification](authentication.md) : Configuration OAuth (Google, Apple).
