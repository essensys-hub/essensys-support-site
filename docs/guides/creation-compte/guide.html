En bref

Pour piloter votre installation à distance (volets, éclairage, scénarios…), vous devez :

  1. Créer un compte sur www.essensys.fr
  2. Vous connecter puis ouvrir le portail domotique sur mon.essensys.fr
  3. Envoyer une demande de liaison avec le n° de série de votre armoire
  4. Nous communiquer dans le message : l’adresse MAC de l’armoire et, si possible, la clé produit
  5. Attendre la validation par l’équipe Essensys (généralement sous 48 h ouvrées)

Où trouver ces informations sur votre armoire ?

1. N° de série et clé produit

Ouvrez la porte de l’armoire électrique. Sur l’étiquette blanche Schneider Electric, notez :

  • N° de série — identifiant unique de votre machine
  • Clef produit — code de licence (5 groupes de caractères)

2. Adresse MAC Ethernet

Sur l’écran tactile de l’armoire, accédez au menu Réglage accès à distance. En bas de l’écran, l’adresse MAC est affichée après @ MAC.

Les voyants CABLE → DHCP → DNS → SERVEUR indiquent l’état de la connexion Internet. Si SERVEUR est rouge avant validation de votre compte, c’est normal : il passera au vert une fois votre liaison approuvée.

Étape 1 — Créer votre compte

  1. Ouvrez votre navigateur et rendez-vous sur
    https://www.essensys.fr/register
  2. Remplissez le formulaire : prénom, nom, e-mail et mot de passe (8 caractères minimum).
  3. Cliquez sur S’inscrire, puis connectez-vous.

Étape 2 — Se connecter

Allez sur https://www.essensys.fr/login et saisissez votre e-mail et mot de passe.

Étape 3 — Ouvrir le portail domotique

Une fois connecté, ouvrez :

https://mon.essensys.fr

Si vous n’êtes pas encore connecté, le portail vous invite à vous identifier :

Étape 4 — Envoyer la demande de liaison

Après connexion, remplissez le formulaire :

  • N° série machine — obligatoire (voir étiquette armoire)
  • Message — recommandé : indiquez la MAC, la clé produit et votre adresse
Exemple de message à copier :
MAC armoire : 00:11:22:33:44:55 — Clé produit : AB12C-DE345-F6789-01234-56789 — Installation : Lyon 69003

Étape 5 — Attendre la validation

Après envoi, l’écran affiche Demande en attente. Un administrateur Essensys vérifie vos informations et approuve la liaison.

  • Rechargez la page (Rafraîchir le statut) après réception de notre confirmation par e-mail
  • Une fois validé, le voyant SERVEUR sur l’armoire doit passer au vert
  • Vous accédez alors aux commandes domotiques depuis le portail

Besoin d’aide ?

Contactez le support Essensys en précisant :

  • Votre e-mail de compte
  • Le n° de série machine
  • L’adresse MAC
  • Une capture de l’écran « Réglage accès à distance » si le voyant SERVEUR reste rouge après validation
